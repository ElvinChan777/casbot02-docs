# 传感器

CASBOT02 搭载多种传感器，提供全方位的环境感知能力。

## 传感器一览

| 传感器 | 数量 | 位置 | 数据类型 |
|---|---|---|---|
| RGBD 相机 | 3 | 头部/胸口/腹部 | 彩色图 + 深度图 |
| 彩色双目相机 | 1 | 头部 | 彩色图 |
| 鱼眼相机 | 2 | 前/后方 | 彩色图 |
| 激光雷达 | 1 | 腹部（速腾聚创 Airy） | PointCloud2 |
| IMU | 1 | 机体 | 姿态 + 角速度 + 加速度 |
