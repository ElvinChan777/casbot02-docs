# 系统概述

CASBOT02 是一款通用人形机器人，身高 160cm，体重 50kg。本页介绍其系统架构和核心参数。

## 系统架构

CASBOT02 采用多模块架构，核心模块包括：

| 模块 | 说明 |
|---|---|
| **Orin 主控** | 运行 ROS2，承载上层应用和二开程序 |
| **HRU（人形机器人单元）** | 负责底层运动控制、传感器采集 |
| **导航控制模块** | 激光雷达接入、导航算法运行 |
| **头显模块** | LED 面部表情显示（UART+RS485） |

!!! tip "硬件通信拓扑"
    Orin 主控与导航控制模块通过以太网交换机连接在同一局域网内。

## 核心参数

| 参数 | 值 |
|---|---|
| 身高 | 160 cm |
| 体重 | 50 kg |
| 操作系统 | Ubuntu 22.04 + ROS2 Humble |
| 主控 | NVIDIA Jetson Orin |
| 自由度 | 41+ DOF（含灵巧手） |

## 关节总览

| 部位 | 关节数 | 说明 |
|---|---|---|
| 头部 | 2 | 偏航 + 俯仰 |
| 腰部 | 1 | 偏航 |
| 单臂 | 7 | 肩3 + 肘1 + 腕3 |
| 单手 | 6 | 拇指2 + 四指各1（主动关节） |
| 单腿 | 6 | 髋3 + 膝1 + 踝2 |
| **合计** | **41** | 不含被动关节 |
